# projeto-ml-vendas
# 📊 Projeto de Machine Learning - Previsão de Vendas

Este projeto implementa um modelo simples de Machine Learning para prever o valor de vendas com base na quantidade de cupons.

---

## 🚀 Objetivo

Criar um modelo capaz de prever o **valor de venda (R$)** a partir de dados históricos de vendas.

---

## 🧠 Tecnologias utilizadas

- Python
- Pandas
- Scikit-learn
- Joblib

---

## 📂 Estrutura do projeto
.
├── dados.csv # Base de dados de treino
├── train.py # Script de treinamento do modelo
├── predict.py # Script de previsão
├── modelo.pkl # Modelo treinado
└── README.md


---

## 📊 Dados utilizados

O modelo utiliza as seguintes variáveis:

- Rede
- Mês/Ano
- Quantidade de vendas (cupons)
- Valor de venda (target)
